How it works

02

Get API access
Create a client + event type 
# 1. Create a client account
POST /v1/users
x-admin-secret: your_secret

{
  "name": "Acme Corp",
  "email": "hello@acme.com",
  "slug": "acme"
}

# → Returns api_key for the client

POST /v1/event-types
x-api-key: client_key

{
  "title": "Discovery Call",
  "slug": "discovery",
  "duration_minutes": 30
}

# → Book at /book/acme/discovery
